== Description ==

Plugin will create input mask and Ukraine phones validation for Woocommerce billing phone field.


Plugin Features

1. Checkout Phone Field will have placeholder with format +38(___) ___-__-__

2. Checkout Phone Field will be mask with format +38(___) ___-__-__

3. Checkout Phone Field will be validated for phone number with format +38(___) ___-__-__


== Installation ==

Upload 'wc-ukr-billing-phone-validation' to the '/wp-content/plugins/' directory and activate plugin.

= Minimum Requirements =

* PHP 7.2 or greater is recommended
* MySQL 5.6 or greater is recommended
